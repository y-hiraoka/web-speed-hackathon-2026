# kuromoji の辞書ファイル（17MB）が常にバンドルに含まれる

## カテゴリ

クライアント / ビルド

## 影響するメトリクス

LCP / TBT

## 影響度

中

## 作業規模

S

## 問題の原因

日本語形態素解析ライブラリ `kuromoji` の辞書ファイル（12ファイル、合計約17MB gzip済み）が `public/dicts/` に配置されている。kuromoji は検索機能でのみ使用されるが、ライブラリ本体は初期バンドルに含まれる。

辞書のロード時にはメインスレッドで重い初期化処理が行われる。

## 原因箇所

`application/public/dicts/` — 12個の辞書ファイル（合計約17MB）
`application/client/webpack.config.js:101-102` (kuromoji の alias 設定)

## 解決方法

1. kuromoji を `import()` で遅延ロード（検索ページ表示時のみ）
2. 辞書の初期化を Web Worker で行う
3. あるいは検索のトークナイズをサーバーサイドに移行（サーバーで形態素解析して結果を返す）
4. サーバーサイド検索に移行すれば kuromoji + bayesian-bm25 をクライアントから完全に削除可能
